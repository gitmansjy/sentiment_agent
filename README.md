# 清洁电器舆情分析系统 🧹

基于 LangChain + 通义千问 + Streamlit 构建的智能舆情分析平台，专注于**扫地机、洗地机、吸尘器**等清洁电器的小红书/电商评论深度挖掘。

---

## ✨ 功能特性

- **多维度情感分析**  
  针对不同产品类型（扫地机/洗地机/吸尘器）自动切换分析维度，如清洁效果、续航、噪音、避障、自清洁等。

- **可解释性 AI**  
  每条评论分析均附带证据片段，展示模型判断的依据，让结果可信可追溯。

- **实时舆情监控与预警**  
  自动检测负面率异常波动，通过看板或 Webhook 推送预警信息，并生成回复草稿。

- **趋势预测**  
  基于 Prophet 时间序列模型预测未来情感走势，帮助提前应对。

- **知识图谱支持（可选）**  
  将评论中提取的产品部件、问题实体存入 Neo4j，挖掘问题间的深层关联（如“电池问题”常与“续航短”同时出现）。

- **可视化看板**  
  集成 Streamlit 构建交互式仪表盘，支持日期筛选、情感分布、方面评分雷达图、词云、评论明细搜索等功能。

---

## 🛠️ 技术栈

| 组件 | 技术 |
|------|------|
| 后端框架 | Python + LangChain |
| 大模型 | 通义千问（Dashscope） |
| 数据库 | MySQL（关系数据）、Neo4j（知识图谱，可选） |
| 预测引擎 | Prophet |
| 可视化 | Streamlit + Plotly + WordCloud |
| 并发处理 | ThreadPoolExecutor（批量分析优化） |
